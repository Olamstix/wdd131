Budget App Site Plan

MiBudget

This name was selected because it conveys personal budget management, which is the core function of the app.

Site Purpose

The purpose of the BudgetPro site is to provide users with essential financial management tools designed to help them effectively manage their finances. It will offer the following features:

  • Dashboard: Overview of income and expenses
  • Expense Tracker: Tool to log and categorize expenses
  • Reports: Visual representations of financial data, such a chart

Scenarios

  • How can I track my daily expenses?
  • What does my monthly expenses look like?

Color Schema

  • Primary Color: #ffffff (used for headings, buttons, and links)
  • Secondary Color: #554994 (used for the background, footer, and accents)

Typography

  • Primary Font: Gilroy (used for body text)
  • Secondary Font: Georgia, serif (used for headings)
